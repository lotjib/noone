## Как запустить

- git clone ...
- composer install
Для скачивания вендора
- npm install
Для фронта (у нас его нет)
- php artisan key:generate
Для генерации ключа
- php artisan migrate
Для миграции БД
- php artisan db:seed
Для ввода тестовых данных
- php artisan serve
Для запуска локального сервера
- php artisan queue:work
Для запуска Очереди
